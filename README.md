This repository is a **feature-rich fork** of **`pokeemerald-expansion`** that serves as both a development base for ROM hackers and the foundation for an ambitious **four-region ROM hack**. Built upon [RHH's pokeemerald-expansion](https://github.com/rh-hideout/pokeemerald-expansion/), [pret's pokeemerald](https://github.com/pret/pokeemerald), and contributions from the wider ROM hacking community.

## üåü What Makes This Repository Special

- **Multiple Feature Branches**: Organized development with specialized branches for different features and improvements
- **Four-Region ROM Hack**: An ambitious project expanding beyond the traditional single-region experience
- **Enhanced Documentation**: Comprehensive [wiki tutorials](docs/tutorials/wiki_index.md) covering everything from basic content addition to advanced system customization
- **Community Contributions**: Integration of features and improvements from various talented ROM hackers and developers
- **Modern Scripting Support**: Full integration with [Poryscript](README_poryscript.md) for cleaner, more maintainable event scripting
